# SST Prime 4.0 — Central Técnica

Versão simplificada e otimizada, sem Assistente IA e sem Conversor PDF/Word.

## Recursos mantidos

- Visão geral e indicadores
- Catálogo de EPIs com busca, filtros, favoritos e exportação CSV
- Riscos e conclusões técnicas
- Empresas e exposições
- Gerador de frases
- Plano de ação 5W2H
- Normas Regulamentadoras
- Painel técnico
- Tema claro/escuro
- Instalação como PWA e funcionamento offline

## Publicação

Publique todos os arquivos desta pasta no GitHub Pages, Netlify ou outro servidor estático. Não é necessário backend, Render, chave de API ou instalação de dependências.
